# Jurni Travel

## 1. Overview
**Jurni** is a full-stack web application for online tour booking and tour management.  
The system allows users to explore tours, view detailed information, and place bookings, while administrators can manage tours, orders, and users efficiently.

This project is reused and further developed for the **Application Development Language** course, focusing on modern web technologies, scalable architecture, and real-world development practices.

---

## 2. Project Objectives
- Apply modern application development languages and frameworks  
- Build a scalable, maintainable, and production-oriented web system  
- Integrate real-time features and AI capabilities  
- Practice teamwork, clean code, and Git-based source control  

---

## 3. Main Features
- Browse and search travel tours  
- View detailed tour information  
- Online tour booking  
- User authentication and authorization  
- Admin dashboard for tour and booking management  
- Real-time notifications and updates  
- AI-powered features for intelligent support and content assistance  

---

## 4. Technology Stack

### Frontend
- React  
- Vite  
- TailwindCSS  
- Clerk (Authentication & User Management)

### Backend
- Node.js  
- Express.js  
- Sequelize ORM  
- MySQL  

### Authentication
- Clerk (Full-stack authentication)

### Real-time Communication
- Socket.io  

### AI Integration
- Google Gemini AI  

### Media Storage
- Cloudinary  

---

## 5. System Architecture
- Frontend and Backend are separated following a **client–server architecture**  
- RESTful APIs are used for data communication  
- Real-time features are implemented using Socket.io  
- Authentication is handled centrally by Clerk across the full stack  
- Media assets are stored and managed via Cloudinary
